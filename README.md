# Tobin Carlton Van Buizen - Academic Portfolio

Welcome to the repository for my personal academic and professional website. 

You can view the live site here: **[tobincarltonvanbuizen.com](https://tobincarltonvanbuizen.com)**

## About This Project
This repository contains the source code for my personal portfolio. It is designed with a "Modern Minimalist Academic" aesthetic to cleanly present my background in mathematics and statistics, alongside my research projects, academic talks, and personal interests. 

As a First Class Honours graduate from the Australian National University (ANU) transitioning into PhD programs, this site serves as a central hub for my academic CV, industry resume, and thesis work in equivariant stable homotopy theory and twisted topological cyclic homology.

## Features
* **Academic Design:** A clean, high-contrast, distraction-free layout tailored for academic reading.
* **MathJax Integration:** Natively renders LaTeX equations (\( C_n \), \( TC_{C_n} \), etc.) directly in the browser for research abstracts and talk descriptions.
* **Centralized CSS:** Uses a single `style.css` file for cohesive theming across all pages.
* **Responsive Layout:** CSS Flexbox ensures the site scales elegantly from desktop monitors to mobile screens.
* **Secure Document Hosting:** Resumes and research papers (PDFs) are hosted directly within the repository and served seamlessly.

## Tech Stack
* **HTML5** (Semantic structure)
* **CSS3** (Custom styling, Flexbox layout, no external frameworks)
* **MathJax** (LaTeX compilation)
* **GitHub Pages** (Hosting & Deployment)

## Repository Structure
* `index.html` - Homepage and biography
* `research.html` - Research projects, publications, and abstracts
* `talks.html` - Academic and public speaking engagements
* `fun.html` - Personal interests (Powerlifting, etc.)
* `style.css` - Global stylesheet
* `academic-cv.html` / `industry-cv.html` - Dedicated document viewer pages
